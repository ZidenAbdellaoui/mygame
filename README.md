# RPG129774 — Classic Fantasy RPG

A turn-based classic fantasy role-playing game developed in Java with JavaFX for the *Metodologie di Programmazione* course at Unicam (AA 2025/26).

## Description

Embark on an adventure as a Warrior, Mage, Rogue, or Paladin. Explore dungeons, fight monsters in turn-based combat, manage your inventory, and complete quests. Your progress is automatically saved as JSON files.

## Requirements

- Java 17 or higher
- Internet connection on first run (Gradle downloads dependencies automatically)

## How to Run

**1. Build:**
```bash
./gradlew build
```

**2. Run:**
```bash
./gradlew run
```

On Windows use `gradlew.bat` instead of `./gradlew`.
